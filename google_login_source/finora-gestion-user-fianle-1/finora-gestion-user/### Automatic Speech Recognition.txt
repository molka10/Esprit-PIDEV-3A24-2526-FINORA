```
import os from huggingface_hub import InferenceClient client = InferenceClient( provider="hf-inference", api_key=os.environ["HF_TOKEN"], ) output = client.automatic_speech_recognition("sample1.flac", model="openai/whisper-large-v3")
```

### Chat Completion (LLM)
Find out more about Chat Completion (LLM) [here](https://huggingface.co/docs/inference-providers/en/tasks/chat-completion).
Language
Client
Provider
Settings

```
import os from openai import OpenAI client = OpenAI( base_url="https://router.huggingface.co/v1", api_key=os.environ["HF_TOKEN"], ) completion = client.chat.completions.create( model="katanemo/Arch-Router-1.5B:hf-inference", messages=[ { "role": "user", "content": "What is the capital of France?" } ], ) print(completion.choices[0].message)
```
